## Conventions

- All prices in **euros (€)** unless stated.
- Technical content in **English**; legal/business French terms kept in French where they have no clean English equivalent (CGV, mentions légales, auto-entrepreneur, franchise en base de TVA).
- Every external link is live at time of writing (16 April 2026). Revisit before relying on a specific price.
- "Sources:" blocks at the bottom of each section cite where info came from.
