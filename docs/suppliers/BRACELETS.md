## Red Flags

- If they ask to pay outside Alibaba → hard no. Always use Trade Assurance.
- Silver feels too light → may be silver-plated brass (not real 925).
- Engraving rough or off-center → request redo or change supplier.
- Tracking number never activates → escalate via Alibaba dispute.
