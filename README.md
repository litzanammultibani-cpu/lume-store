# LETHE - Fine Silver Jewelry

**The Art of Timelessness**

LETHE is a fine sterling silver jewelry brand born in Strasbourg, France. Each piece is designed with intention -- timeless silver jewelry made to last and meant to go everywhere with you.

## Collections

- **Essential** -- Minimalist everyday pieces
- **Signature** -- Statement chains and cuffs
- **Eclat** -- Limited edition luxury

## Tech Stack

This is a static website built with:

- HTML5
- CSS3 (responsive, mobile-first)
- Vanilla JavaScript (scroll animations, interactions)
- Google Fonts (Cormorant Garamond, Inter)

## Running Locally

Using npm (requires Node.js):

```bash
npm start
```

Or with Python:

```bash
python -m http.server 8080
```

Then open http://localhost:8080 in your browser.

## Deployment

This site is ready for deployment on:

- **GitHub Pages** -- Push to a repo and enable Pages from Settings
- **Netlify** -- Drag and drop the folder, or connect the repo
- **Vercel** -- Import the repo for instant deployment
